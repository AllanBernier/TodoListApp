### Simple Trello APP in Express@4.19 Angular@18 (Node@22) ###

Installation & usage
```bash
git clone https://github.com/AllanBernier/TodoListApp.git
cd TodoListApp
```

### Backend (Terminal 1)
```bash
cd backend
npm install
npm start
```

### Frontend (Terminal 2)
```bash
cd frontend
npm install
ng serve
```

### Open your browser on http://localhost:4200/login

### Features
- User registration
- User login
- User logout

- Create a new board
- Edit a board
- Delete a board

- Create a new list
- Delete a list

- Create a new card
- Move a card to another list

### Tips:
- Don't use angular (https://npmtrends.com/angular-vs-react-vs-vue)
