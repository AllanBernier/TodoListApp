### Simple Trello APP in Express@4.19 Angular@18 (Node@22) ###

Installation & usage
```bash
git clone https://github.com/AllanBernier/TodoListApp.git
cd TodoListApp
```

### Backend (Terminal 1)
```bash
cd backend
npm install
npm start
```

### Frontend (Terminal 2)
```bash
cd frontend
npm install
ng serve
```

### Open your browser on http://localhost:4200/login
